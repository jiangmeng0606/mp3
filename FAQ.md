# 常见问题解答 (FAQ)

## GitHub Copilot 计费问题

### Q: GitHub Copilot 中的 Gross amount（总金额）需要在月付费之外另外支付吗？

**不需要。**

GitHub Copilot 账单页面中显示的 **Gross amount（总金额/税前金额）** 是您订阅费用的税前价格，并不是在月度订阅费之外需要额外支付的金额。

具体说明：

- **Gross amount**：指未扣税之前的订阅费用原价，即 GitHub Copilot 的订阅金额本身。
- **税费（Tax）**：部分地区会根据当地法律在 Gross amount 基础上加收税费。
- **实际支付金额**：= Gross amount + 适用税费（如有）。

因此，Gross amount **不是**额外收取的费用，它就是您月度或年度订阅的基础价格。如果您所在地区不需要缴税，则 Gross amount 即为您实际支付的全部金额。

如需了解更多计费信息，请参阅 [GitHub Copilot 官方定价页面](https://github.com/features/copilot#pricing)。
